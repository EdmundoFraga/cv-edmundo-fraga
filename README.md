# Currículum Vitae Online - Edmundo Fraga
Currículum Vitae web desarrollado con HTML5 y CSS puro, sin frameworks, como práctica académica del CFGS ASIR.

# Objetivo
Crear un CV web profesional, accesible, responsive, preparado para impresión y desplegado en Vercel.

# Tecnologías
- HTML5 semántico
- CSS3 (Flexbox, variables, media queries)
- Mobile-first
- Accesibilidad básica
- Deploy en Vercel

# Características
- Diseño responsive (móvil y escritorio)
- Navegación accesible por teclado
- Animaciones CSS con propósito
- Uso de pseudo-elementos
- Modo impresión optimizado con URLs visibles

Puedes ver la versión online aquí: cv-edmundo-fraga.vercel.app
